# open-design 啟動捷徑

## 用法

雙擊 `start.bat`，或在終端機執行：

```
tools\open-design-shortcut\start.bat
```

啟動後自動開啟 http://localhost:3010

## 何時用

- 要做新工具的 UI 設計稿時
- 要做海報、Landing Page、社群圖卡時（取代手工 HTML）
- 需要快速生成多個視覺方向供選擇時

## 實際位置

```
G:\我的雲端硬碟\open-design\   ← 主程式在這裡
```

## 更新

cd "G:\我的雲端硬碟\open-design"
